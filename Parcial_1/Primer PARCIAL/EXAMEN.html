Ecuación cuadrática

Ecuación cuadrática:

Una ecuación de segundo grado o ecuación cuadrática de una variable es aquella que tiene la expresión general:

Ecuación de segundo grado:

ax2 + bx + c = 0, a≠0


Donde x es la variable; mientras que a, b y x son las variables constantes y corresponde con los coeficientes:

a
cuadrático (a)
b
Lineal (b)
c
Independiente (c)

Para una ecuación cuadrática con coeficiente reales o complejos existen siempre 2 soluciones (x1 y x2), no necesariamente distintas, llamadas raíces, que ser reales o complejas.
